1.What is DOM and what is the purpose of the DOM?

Answer: Document object Model (DOM) is an interface to web pages.
It reads the codes and modify the page's content, structure, and styles. When a web page is loaded, the browser creates a Document Object Model of the page.

2.How select Element from DOM?

Answer: getElementsByTagName() ; getElementsByClassName() ; getElementById() ; querySelector() ; querySelectorAll() ;

3.What is event bubble?

Answer: Event bubble is an event which occurs within a nested set of HTML elements is caught by the innermost element and travels up the DOM tree hierarchy to its outermost element.

4.What is call back function and its usability?

Answer: A callback function is a function which is passed as an argument to another function and then executed within the outer function.
Therefore, a callback function is only called once a clearly defined operation has been performed. Event handlers are a good example of a type of callback function. Other well-known programming languages (Java, PHP, and Python) also use callbacks to easily pass specific user parameters.
